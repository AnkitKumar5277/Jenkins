CI/CD 
(Continuous Integration/Continuous Deployment) 
ek software development practice hai jisme code changes ko regularly aur automatically build, test, aur deploy kiya jata hai.

Haan bhai, Jenkins ko seekhna software tester ke liye zaroori hai kyunki:
Ye automated test suites ko run karta hai.
Web aur mobile automation suite ko bhi chala sakta hai.

Report aur email notification bhejta hai.
